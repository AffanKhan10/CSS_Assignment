1. What are the main differences between external, internal, and inline CSS?

External CSS is defined in a separate file and linked to the HTML document, while internal CSS is defined within the head section of the HTML document. Inline CSS is defined within a specific HTML tag using the style attribute. External CSS is more efficient for styling larger websites as it allows for greater consistency and easier maintenance, while internal and inline CSS are more suitable for smaller projects or individual tags.

2. What is the syntax for class and ID selectors?

The syntax for a class selector is a period followed by the class name, e.g. .classname. The syntax for an ID selector is a pound sign followed by the ID name, e.g. #idname.

3. How would you apply a single rule to two different selectors?

You can separate the selectors with a comma, e.g. selector1, selector2 { rule: value; }

4. Given an element that has an id of title and a class of primary, how would you use both attributes for a single rule?

#title.primary { rule: value; }

5. What does the descendant combinator do?

The descendant combinator, represented by a space between selectors, selects elements that are descendants of another element. For example, "parent selector descendant selector" selects all descendants of the parent selector.

6. Between a rule that uses one class selector and a rule that uses three type selectors, which rule has the higher specificity?

The rule that uses the class selector has higher specificity than the rule that uses three type selectors.

7. From inside to outside, what is the order of box-model properties?

The order of box-model properties from inside to outside is padding, border, margin.

8. What does the box-sizing CSS property do?

The box-sizing property determines how the width and height of an element is calculated. The default value is content-box, which only includes the content area in the width and height calculation. The border-box value includes the padding and border in the calculation, resulting in a box with a fixed size.

9. What is the difference between the standard and alternative box model?

The standard box model calculates the width and height of an element based on the content box, padding, border, and margin. The alternative box model, using the box-sizing property with a value of border-box, calculates the width and height of an element based on the content box and padding, but includes the border and margin in the overall size of the box.

10. Would you use margin or padding to create more space between 2 elements?

You would use margin to create space between two elements.

11. Would you use margin or padding to create more space between the contents of an element and its border?

You would use padding to create space between the contents of an element and its border.

12. Would you use margin or padding if you wanted two elements to overlap each other?

If you want two elements to overlap each other, you would use negative margin. By applying a negative margin to one or both elements, you can make them overlap. Padding, on the other hand, adds space between an element's content and its border, and would not make two elements overlap.

13. What is the difference between a block element and an inline element?

Block elements take up the full width available and create a new line after the element, while inline elements only take up the necessary width for the content and do not create a new line after the element.

14. What is the difference between an inline element and an inline-block element?

An inline-block element has characteristics of both a block and inline element. It takes up only the necessary width for the content like an inline element, but also allows for height and width properties to be set like a block element.

15. Is an h1 block or inline?

An h1 element is a block element.

16. Is button block or inline?

A button element can be either a block or inline element depending on the CSS styling applied.

17. Is div block or inline?

A div element is a block element.

18. Is span block or inline?

A span element is an inline element.

19. What’s the difference between a flex container and a flex item?

A flex container is an element that contains flex items and is styled with the display:flex property. A flex item is an element that is a direct child of a flex container and can be styled using flex properties such as flex-grow, flex-shrink, and flex-basis.

20. How do you create a flex item?

To create a flex item, simply include an element as a direct child of a container with the display:flex property.

21. What are the 3 values defined in the shorthand flex property?

The shorthand flex property includes the values for flex-grow, flex-shrink, and flex-basis. The values can be defined in the following format: flex: [flex-grow] [flex-shrink] [flex-basis].

22. How do you make flex items arrange themselves vertically instead of horizontally?

You can use the flex-direction property set to column to make flex items arrange themselves vertically.

23. What is the difference between justify-content and align-items?

justify-content aligns flex items horizontally, while align-items aligns flex items vertically.

24. How do you use flexbox to completely center a div inside a flex container?

You can center a div inside a flex container by setting the flex container's properties to display:flex, justify-content:center, and align-items:center.

25. What’s the difference between justify-content: space-between and justify-content: space-around?

The justify-content: space-between property evenly spaces flex items with no space at the beginning or end. The justify-content: space-around property evenly spaces flex items with equal space around each item, including space at the beginning and end.
